## maptalks-plugin-cli

> A simple cli scaffolding for maptalks plugins

### Quickstart

```bash
npm install -g maptalks-plugin-cli
# or
yarn global add maptalks-plugin-cli

// list available templete
maptalks-list
// generate project
maptalks-init maptalks-plugin-tpl-base project
cd project
npm i
npm run dev
```
